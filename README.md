# template_python_project_structure
A template of the folder structure that is recommended for use with all One Epsilon python projects

## Getting started
1. To get started, first make sure python is installed on your machine. Python 3.14 is recommended.
2. Open up a new termal using `Ctrl + Shift + '`.
3. Create a new local python environment by typing ```python -m venv venv``` into the terminal.
4. Press `Ctrl + Shift + P` and type `Python: Select Interpreter`, then select "venv" to activate your local environment.
5. Open up a new terminal `Ctrl + Shift + '`. You should see that the terminal text has '(venv)' in it.
6. Add any required python packages that you may need to the list of dependencies in the 'pyproject.toml' file.
6. Install you local python package and your required dependencies using ```pip install -e .```
